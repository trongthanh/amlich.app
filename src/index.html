Âm Lịch Việt Nam

Dành cho terminal & AI agent

Cập nhật 04-2026: Mình đã thêm tính năng trả về thông tin ngày Âm dưới định dạng markdown tiết kiệm token dành cho AI agents và ANSI color dành cho terminal.

Truy cập amlich.app trực tiếp qua curl, wget hoặc AI agent — không cần trình duyệt: 

curl -L amlich.app               # hôm nay / today
curl -L amlich.app/2025-06-15    # dương lịch / solar date
curl -L amlich.app/L2025-05-20   # âm lịch / lunar date

AI agents (Claude, GPT, v.v.) tự động nhận kết quả dạng Markdown. Xem thêm tại /llms.txt.
